## 🚀 Setup & Launch / O'rnatish va Ishga Tushirish

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

Tizimingizda [Node.js](https://nodejs.org/) o'rnatilganligiga ishonch hosil qiling.

### 1. Set Active Workspace / Faol Ishchi Sohani Sozlash
We recommend opening the project folder as your main editor workspace:
```bash
C:\Users\zed\.gemini\antigravity\scratch\texas-burger-about
```

### 2. Install Dependencies / Kutubxonalarni O'rnatish
Open your terminal inside the project directory and run:
```bash
npm install
```

### 3. Run Development Server / Loyihani Ishga Tushirish
Start the local Vite server:
```bash
npm run dev
```
Open your browser and navigate to the local URL (usually `http://localhost:5173`) to explore the gorgeous 3D experience!

### 4. Build for Production / Prodyukshn uchun Yig'ish
Compiles and optimizes the project for hosting:
```bash
npm run build
```
